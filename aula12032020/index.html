QUAIS AS FUNÇÕES DE CADA TAG?

TAGS FUNÇÕES
p parágrafo
h1,h2,h3,h4,h5,h6 título
b deixa texto em negrito
br / quebra parágrafo
strong negrito versão html5
i letra em italico
hr / cria uma linha
ul lista não ordenada
ol lista ordenada
li cria um item para lista
u texto sublinhado
!- cria um comentário
html define um documento html
head envolve todo ocabeçalho
meta define informações gerais
style informações de estilo
title define título
script define linguagem script
a hiperligação,indicar um link
input campo para inserir texto
center centralizar
select lista selecionavel
td cria uma célula de tabela
th cabeçalho de uma tabela
tr cria uma linha em uma tabela

O que são?

  • ID

    • usado para identificar um único elemento

  • Style

    • permite colocar elementos do CSS, usado para personalizar sua página

  • Class

    • identificar grupos de elemento
