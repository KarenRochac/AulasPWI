QUAIS AS FUNÇÕES DE CADA TAG?

TAGS FUNÇÕES
p parágrafo
h1,h2,h3,h4,h5,h6 título
b deixa texto em negrito
br / quebra parágrafo
strong negrito versão html5
i letra em italico
hr / cria uma linha
ul lista não ordenada
ol lista ordenada
li cria um item para lista
u texto sublinhado
!- cria um comentário

O que são?

  • ID

    • usado para identificar um único elemento

  • Style

    • permite colocar elementos do CSS, usado para personalizar sua página

  • Class

    • identificar grupos de elemento
